# uc11-leiloes

Atividade 1 UC11

O projeto se trata de um teste de repositório github, e a sincronização de commits de um arquivo sql entre GitBash e GitHub, e conectar os dois repositórios(local e remoto/online) além de criar um README.md legível e explicativo

Foi utilizado nesse projeto o MySQL para o arquivo em questão que foi posto em commit, e o GitBash para os comandos.
